Teaching Portfolio
==================

## Theme

Based on <https://hyde.getpoole.com/>.


## Install

Install Ruby 2.5+:

    sudo apt-get install build-essential ruby ruby-dev
    ruby -v

Install bundler:

    sudo gem install bundler

Install required packages:

    bundle install


## Build and serve

Serve locally at <http://localhost:4000/>:

    bundle exec jekyll serve
